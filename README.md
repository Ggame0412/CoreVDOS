# CoreVDOS

**CoreVDOS** — это высокопроизводительная программируемая платформа-песочница (Sandbox Runtime), спроектированная по канонам оптимизации 2000-х годов. Главная цель движка — выжимать максимальный FPS на слабых устройствах (включая Android и ноутбуки эпохи 2011 года) при симуляции сложных миров, управляемых текстовым DSL.

Вместо интерпретации тяжелых скриптов «на лету», CoreVDOS использует собственный компилируемый язык описания реальности **VDOSL**, полностью исключая задержки (Stuttering) и микро-фризы.
